Problem Definition

핵심 난제와 공략 전략

작은 객체·유사 클래스 혼동·누락 문제를 3축 전략으로 해결했습니다.

Challenges

Small Object Detection

낮은 해상도와 복잡한 배경으로 인해 작은 알약 객체의 BBox 누락(False Negative) 빈번 발생

Class Confusion

형태와 색상이 매우 유사한 클래스 간의 오분류(Misclassification) 문제 심각

Multi-Object Balance

한 이미지 내 다수 객체(3~4개) 존재 시, 오탐(FP)과 미탐(FN)의 트레이드오프 관리 어려움

Breakthroughs

Data-Centric Approach

정제 규칙(Rule-based) 강화 및 외부 데이터 통합으로 입력 품질 극대화 (Quality > Quantity)

Systematic Experiment

Spacer

가설-검증-회고 루프의 체계적 운영으로 최적의 하이퍼파라미터 및 모델 선정

Advanced Post-processing

Top-4 강제 선택 로직 및 Confidence Threshold 튜닝으로 Precision/Recall 최적 균형점 도달
